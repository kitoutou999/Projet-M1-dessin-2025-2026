# Projet Dessin — M1 Design Patterns

Academic project (M1) — a Java drawing application built to illustrate and implement classic design patterns.

## Design Patterns Implemented

| Pattern | Usage |
|---|---|
| **Command** | Undo/redo operations on shapes |
| **Composite** | Grouping shapes into hierarchies |
| **Observer / MVC** | UI updates on model change |
| **Strategy** | Interchangeable drawing algorithms |
| **State** | Mode switching (normal, hard, two-player) |
| **Factory Method** | Shape creation |

## Build & Run

```bash
./compile.sh
./run.sh
```
